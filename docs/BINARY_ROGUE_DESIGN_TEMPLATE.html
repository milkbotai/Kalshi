Quick Start Checklist

Complete these steps when starting a new Binary Rogue site.

1. Copy binary-rogue.css to your project
2. Include Google Fonts (Inter + JetBrains Mono)
3. Add Freckle Face font locally
4. Set {{SITE_NAME}} and {{SITE_SUBTITLE}}
5. Choose your tagline
6. Select card variants for your use case
